`kelunik/rate-limit` is a non-blocking rate limiting library for use with the [`amp`](https://github.com/amphp/amp) concurrency framework.

**Required PHP Version**

- PHP 7.0+

**Installation**

```bash
$ composer require kelunik/rate-limit
```
